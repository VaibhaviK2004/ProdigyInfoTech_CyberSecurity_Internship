# 🖼️ Task 02 – Image Encryption with Pixel Manipulation

This notebook demonstrates a simple image encryption and decryption system using XOR-based pixel operations. Built as part of my Cyber Security Internship with [Prodigy InfoTech](https://prodigyinfotech.dev).

## ✅ Features

- Load and visualize images
- Encrypt using XOR operation
- Decrypt using the same key
- Display all stages using Matplotlib

## 🛠️ Tools Used

- Python
- Pillow (PIL)
- Matplotlib
- NumPy
- Jupyter Notebook

## 📸 Sample Output

- Original Image
- Encrypted Image
- Decrypted Image (restored)
